Detailed Description

template<class T>
class minerva::Tensor< T >

Tensor managing memory on the GPU presented as an array of type T.

Template Parameters
Telement type of the array.

Tensor manages a storage buffer object on the GPU. In the shader these can be defined like in the following:

layout(std430, binding = 0) buffer tensor { float x[]; };

The benefit here is, that the shader does not need to know the actual size of the SBO. This should be used for large data.

To fill the tensor you need to copy it from a Buffer using a command issued to a sequence. Likewise, to retrieve the data, you also need a command:

Buffer<float> buffer(context, {1.f, 2.f, 3.f, 4.f});
Tensor<float> tensor(context, 4);
auto copyTo = updateTensor(buffer, tensor);
auto copyFrom = retrieveTensor(tensor, buffer);
Timeline timeline(context);
beginSequence(timeline)
.And(copyTo)
.Then(copyFrom)
.Submit();
