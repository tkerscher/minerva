Detailed Description

Uniform manages a uniform buffer object on the GPU, which is mapped on the CPU side to allow both reads from the GPU and updates from the CPU. UBOs can be defined in the shader like the following:

layout(std140, binding = 0) uniform UBO {
float someFloat;
int someInt;
float anotherFloat;
} ubo;

Interacting with this UBO can look like this:

struct UBO {
float someFloat;
int someInt;
float anotherFloat;
};
Uniform<UBO> ubo(context);
ubo->someInt = 4;
