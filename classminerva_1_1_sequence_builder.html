Detailed Description

Builder class for creating a command sequence.

minerva::SequenceBuilder allows to create a sequence of commands running groups of commands after each other, while the commands in the group runs in parallel. The groups can be build using minerva::SequenceBuilder::And and a new group waiting on the previous one can be issued with minerva::SequenceBuilder::Then or with minerva::SequenceBuilder::WaitFor one that wait for an external set step.
