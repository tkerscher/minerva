Detailed Description

Buffer specialized for interaction with Image.

ImageBuffer is a specialization of Buffer providing additional functions for interacting with images on the filesystem. Internally it utilizes stb_image and thus provides for example the following formats for loading:

  • PNG
  • JPG
  • BMP
  • GIF
  • PPM

While it will only save images as png.

Images will always be loaded as 4 channel RGBA 8bit per channel image.
