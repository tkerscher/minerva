Detailed Description

template<class T>
class minerva::Dispatch< T >

Struct holding information for a single Program dispatch.

Template Parameters
Ttype of the push constant, void if there is no push constant.

Dispatch holds information for a single shader dispatch. Compute shaders hold the following defining the size of a single work group:

layout(local_size_x = 8, local_size_y = 8) in;

Dispatch now defines how many of these work groups in each dimension.

Additionally, one can also provide push constants. In the shader they look like the following:

layout(push_constant) uniform constants {
vec2 trans;
float scale;
} push;

and on the host side:

struct Push {
float transX;
float transY;
float scale;
};
Push push{};
Dispatch<Push> dispatch{ 16, 16, push };
